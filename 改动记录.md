# DramaBox 改动记录

## 显存自动适配
- `config.py`：启动时检测 GPU 显存，自动选档位
  - >=20G：全 BF16 常驻
  - 10~20G：INT8 + Gemma CPU 搬运
  - 6~10G：INT8 + 编码时 DiT 也卸 CPU
- Test 峰值 15.25G，我们**峰值 ~8G**，8G 卡可跑

## Gemma CPU 搬运
- `inference_server.py`：Gemma 编码完 `.to('cpu')` 存系统内存，下次 `.to('cuda')` 搬回
- 不删不重载，**每次省 7s 磁盘加载时间**
- 超低显存模式（<10G）：编码时连 DiT 也卸到 CPU，峰值只计一个大模型

## 噪音屏蔽
- `config.py`：过滤第三方库 WARNING（transformers/torch/distributed/tqdm）
- `启动.bat`：`PYTHONWARNINGS=ignore` + `TQDM_DISABLE=1`

## UI 重设计
- `app.py`：暗色影棚风，琥珀橙主色，自定义 CSS
- 三 Tab：快速生成 / 音色库 / 对话工坊

## 音色库
- `src/voice_library.py`：上传/命名/列表/删除，JSON 存元数据
- 数据存 `models/voices/` + `models/voices.json`

## 对话工坊
- `src/dialogue.py`：粘贴剧本 → 自动解析角色 → 分配音色 → 一键生成完整对话
- 格式：`角色名：台词`，每行一句
- 逐句生成后自动拼接，加句间停顿

## 提示词示例重写
- `app.py` EXAMPLES：8 条中文参考音频替换原英文音频
- 舞台指示全英文化（模型训练语言），台词保留中文
- 每条例 3-5 层情绪递进，官方仅 2 条各 4 层
- 笑声用 `"Hahaha."` / `"Hehehe."`（单英文词），中文笑声模型会逐字念

## Prompt 快捷插入组件
- `app.py`：8 个说话人标签（温柔女声/霸气男声/御姐冷艳 等），一键插英文描述
- 18 个非语言音效下拉（情绪动作/停顿节奏/嗓音变化），支持 Enter 快插和光标定位
- `「」→""` 按钮：选中中文自动包双引号，无选中时插入 `""` 光标置中
- 纯 JavaScript 内联实现，不依赖额外库，Gradio 兼容

## 环境检查
- `app.py` `_check_environment()`：启动时检测中文路径和 GPU 驱动
- 中文路径拦截：CUDA/PyTorch 不支持非 ASCII 路径，直接报错退出并提示解决方案
- GPU 驱动检测：跑 `nvidia-smi`，未安装则报错并给出驱动下载链接

## 比 Test 强的地方
1. VRAM 峰值低（8G vs 15G），低配卡可跑
2. 自动适配显存，用户不用手动调
3. 音色库管理，不是每次手动上传
4. 对话工坊批量生成，Test 只能一次一段
5. UI 好看，不是默认灰色皮肤
